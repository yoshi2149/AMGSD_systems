# AMGSD_system
メッシュ農業気象データを用いたスイートコーン成長予測システム  
## 機能の概要
### 1. Googleスプレッドシート

**役割：入力インターフェース＋結果表示**

- 緯度・経度・期間・しきい値などのパラメータをユーザーが入力する
- GASを通じて外部API（Render）にデータを送信する起点となる
- APIから返ってきた計算結果（気温、積算温度、降水量など）を可視化・保存する

👉 **現場で使う「操作画面」兼「結果確認画面」**

---

### 2. Google Apps Script（GAS）

**役割：スプレッドシートと外部APIをつなぐ中継・制御層**

- スプレッドシートから入力値を取得し、JSON形式に整形する
- Render上のAPI（Flaskアプリ）にHTTPリクエストを送信する
- 返ってきた計算結果をシートに書き込む
- 複数シートの一括更新などの処理を自動化する

👉 **フロント（シート）とバックエンド（Render）をつなぐ「橋渡し役」**

---

### 3. Render

**役割：バックエンドアプリを公開・実行するクラウド環境**

- GitHub上のコード（app_sweetcorn.py）をデプロイして常時実行する
- 外部（GASなど）からのHTTPリクエストを受け付ける
- APIサーバーとして計算処理を提供する

👉 **計算プログラムをインターネット上で動かす「実行基盤」**

---

### 4. github

**役割：気象データ処理・予測計算のコアロジック**

- GASから送られた入力パラメータを受け取る
- 気温データの取得・整形・積算温度計算などを実行する
- 予測結果（GDD到達日、降水量、履歴データなど）をJSON形式で返す

👉 **実際の「計算・予測」を担う中核プログラム**

<details>
<summary>app_sweet-corn.py</summary>
  **役割：スイートコーン成長予測用プログラム**
  
  + このpythonファイルは指定された緯度・経度の過去値・予報値を元に、スイートコーンの絹糸発生日、収穫日を予測します。
  +  予測手法
    + 閾値を10度とした積算平均気温
  + こちらは旧手法を適用したプログラムとなります。
  + 対応Googleスプレッドシート：temp_app_sweet-corn
    + 対応シート：旧手法版
      + 異なる圃場で使用したい場合、シートをコピーすればそのまま使用できます。
  + 対応GASプログラム：Code.gs
  + 対応Render Service name：AMGSDS-1
</details>
